bilibili将启动“青少年权益保护中心”等一系列保护青少年用户的措施

日常
哔哩哔哩弹幕网
关注
随着互联网的快速发展，当我们在享受互联网带来的便利与乐趣的同时，也有某些不法分子利用互联网侵害青少年。

bilibili作为一个互联网平台，会主动担负起保护和引导青少年的责任。我们将推出一系列措施，防止不法分子危害青少年的成长与权益。

我们将在原有的社区规则（审核、举报、小黑屋、风纪委、众裁等）基础上，建立专门的青少年保护体系，并在未来三个月内全面落实运行。

一、启动“青少年防火墙”计划

1.加强身份认证
我们将在正式会员答题（100题）准入制度的基础上，加强身份认证措施，对于未经身份认证的用户或身份证显示为少年儿童年龄段的用户，我们将用专门的策略进行内容展现和社区权限设置。

加强版身份认证功能计划于4月中旬上线。

 
2. 针对青少年用户优化的内容展现机制
通过官方标签或用户标签等方式，对内容展现机制进行优化。疑似不适合少年儿童年龄段用户在无陪同、无引导等条件下观看的内容，将不对其展现。

对于青少年的引导和保护，是互联网平台应承担的社会责任，我们希望广大用户理解和认同。

注：用户的个人隐私信息均会被严格加密，确保不被泄露。

二、建立“青少年权益保护中心”

1. 我们将与共青团中央等部门共建“青少年权益保护中心”，通过大数据分析、关键词管理等措施，从严识别并过滤对青少年的不良信息。

2. 强化举报入口，针对潜在危害青少年的色情、暴力等不良内容、言论或私聊信息，添加专项举报入口，由“青少年权益保护中心”专项处理，保证速度和力度。

3. 加大对青少年不良信息的处罚力度，一经发现，对稿件和上传者一律执行永久封禁，严重情况将由“青少年权益保护中心”上报有关部门。

4. 特聘法律专家，接受投诉、委托。当发现青少年在bilibili平台疑似受到不法分子侵害或骚扰，“青少年权益保护中心”将为其提供特别法律援助。

5. 特聘青少年心理辅导专家和社会工作专家，为被青少年互联网使用问题所困扰的家庭提供咨询服务，必要时为当事人提供心理辅导介入。

“青少年权益保护中心”计划于5月中旬正式启用。

对青少年的保护将是bilibili的长期工作。我们会跟随时代的发展，不断完善对青少年用户的保护和引导。

青少年保护问题反馈与举报邮箱：teenprotect@bilibili.com
